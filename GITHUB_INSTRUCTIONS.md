# Guía de Carga a GitHub

Has inicializado tu repositorio local correctamente. Para subirlo a GitHub y tener un portafolio profesional, sigue estos pasos:

## Paso 1: Crear el Repositorio en GitHub
1. Inicia sesión en [GitHub.com](https://github.com/).
2. Haz clic en el botón **+** (arriba a la derecha) y selecciona **New repository**.
3. Nombre del repositorio: `gestion-visitas-ventas-gas` (o el que prefieras).
4. **Público**: Selecciona "Public" si quieres que sea parte de tu portafolio visible.
5. **No** selecciones "Initialize this repository with a README" (ya tenemos uno).
6. Haz clic en **Create repository**.

## Paso 2: Conectar y Subir (Push)
Copia los comandos que GitHub te muestra bajo el título "…or push an existing repository from the command line". Deberían ser similares a estos:

En tu terminal (dentro de la carpeta del proyecto), ejecuta:

```bash
# Ya hemos configurado el origen por ti, así que solo necesitas subir:
git push -u origin main
```

**Nota**: El repositorio está configurado con **HTTPS** (`https://github.com/PezcaderiaSAS/gestion-visitas-ventas-gas.git`). Es posible que te solicite tus credenciales de GitHub (Usuario y Token/Contraseña) en una ventana emergente.

Si te pide credenciales, usa tu correo y contraseña (o Token de Acceso Personal si tienes 2FA activado).

## Paso 3: ¡Listo!
Refresca la página de GitHub y verás tu código, el README con las imágenes y la documentación profesional lista para compartir.
